GRID BUILDER A Chrome Extension for Web Developers

Grid Builder is a tool built for front-end developers to inspect, examine and perfect their designs or any web site in action.

DOWNLOAD

Bootstrap Default Mode

When set to "Use Bootstrap Defaults", Grid Builder tracks the window width and adapts to Bootstraps default grid sizing.

Manual Settings

When set to "Use Manual Settings", Grid Builder does not track the window width. Instead it sticks with the given settings whatever the window width is.

Disable If Not Needed

When you're done with analyzing a page, Grid Builder can be disabled fully so that the upper right ON-OFF button would disappear.
